AWS Solutions Architect- Associate

AWS (Amazon Web Services) and Azure (Microsoft Azure) are two major cloud computing platforms that offer a wide range of services to help individuals and businesses deploy and manage their applications and infrastructure in the cloud. The focus of this certification is on the design of cost and performance optimized solutions, demonstrating a strong understanding of the AWS and Azure Well-Architected Framework. The AWS Certified Solutions Architect– Associate certification is designed for individuals who have some experience designing distributed applications and systems on the Amazon Web Services (AWS) platform. It validates a candidate’s ability to architect and deploy secure and robust systems on AWS, as well as their knowledge of best practices in cloud architecture.

Topics Covered through the Course:

Preparing for and attaining this course will showcase:

  • Knowledge and skills in compute, networking, storage, and database AWS and Azure services as well as its deployment and management services
  • Knowledge and skills in deploying, managing, and operating workloads on AWS and Azure as well as implementing security controls and compliance requirements
  • Knowledge and skills in using the AWS and Azure Management Console and the Command Line Interface (CLI) Understanding of the AWS and Azure Well-Architected Framework, networking, security services, and the global infrastructure
  • Ability to identify which AWS and Azure services meet a given technical requirement and to define technical requirements for an AWS and Azure-based application
