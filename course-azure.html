Microsoft Azure Fundamentals

Microsoft Excel is a powerful spreadsheet application that is part of the Microsoft Office suite. It allows users to organize, analyze, and visualize data through a grid of rows and columns. Key features include formulas, pivot tables, and charting tools, which facilitate complex calculations and data analysis.

Topics Covered through the Course:

This course is seperated into 3 parts based on the level of complexity. The following parts are Excel Fundamentals, Intermediate and Advanced. Each part contains the foundational information about excel and its functions. It is important to fully understand and work out each of the functions to get a better understanding of the upcoming concepts. For further queries, please contact through mail or whatsapp.

  2. Module 2:- Core architectural components of Azure.
  3. Module 3:- Azure compute and networking services.
  4. Module 4:- Azure storage services.
  5. Module 5:- Azure identity, access, and security.
  6. Module 6:- Cost management in Azure.
  7. Module 7:- Features and tools in Azure for governance and compliance.
  8. Module 8:- Features and tools for managing and deploying Azure resources.
  9. Module 9:- Monitoring tools in Azure.
